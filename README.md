## Pasos para ejecutar el proyecto
### Backend
activar el entorno virtual con visual studio 

1. pulsar F1
2. Escribir "Python: Select Interprete"


```
cd backend
.\venv\Script\activate
python .\src\app.py
```
o usando docker
```
    docker-compose up 
```

### Frontend
```
cd frontend    
npm i
npm run dev
```
